# Denoising Diffusion Probabilistic Models (DDPM) Implementation

This repository contains a PyTorch implementation of the paper "Denoising Diffusion Probabilistic Models" by Ho et al. (2020).

## Installation

```bash
pip install -r requirements.txt
```

## Usage

### Training
```bash
python train.py --run_name DDPM_Run1
```

### Testing
```bash
python test_implementation.py
```
